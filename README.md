Usage
---------

1. Install
`$ cd ~`
`$ git clone xxx/dotfiles.git .`
3. Set Environment
`$ ./dotfiles/.bin/dotsinstall.sh install`
4. vim plugin install
`$ vi`
`:PlugInstall`
5. exec zsh
`type y`
6. Enjoy!
