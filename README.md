Overview
---------

My home dot files


Usage
---------

1. Download
`$ cd ~`
`$ git clone xxx/dotfiles.git .`
3. Install
`$ ./dotfiles/.bin/dotsinstall.sh`
4. vim plugin install
`$ vi +PlugInstall` or `:PlugInstall` after vim boot
5. zsh plugin install
`$ exec zsh`
6. Enjoy!
